Kamigami Jurassic World • iOS/Android • 12/2017 – 3/2018

Summary

Challenge: Redo the Kamigami Robots app for a Jurassic World walking dinosaur toy. Give the robot more personality and physical interactivity, and increase user engagement with the app. And do it all in ten weeks.

Role: Software team lead, UX designer, iOS developer. I led a team of four developers, plus a couple design contractors who provided artwork.

Process

Pragmatic approach: need to map what app/robot can do now onto achievable set of new features (best bang for the buck).

For interactivity, mapped current behaviors and in-process work to “realistic” dino behaviors.

For engagement: challenges + unlockables. Reuse game engine to build “challenges” — meaning little new code, no new UI, and challenges could be built by non-devs using drag-and-drop UI; add unlockable features and personalizable elements to increase engagement (give users more to do)
